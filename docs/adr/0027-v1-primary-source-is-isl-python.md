# V1 primary source is ISL Python

KnowAct v1 uses *An Introduction to Statistical Learning with Applications in Python* as the primary authoritative source for the `classical_supervised_ml_algorithms` benchmark domain. Candidate nodes should be extracted from its classical supervised learning chapters, while deep learning, unsupervised learning, survival analysis, and multiple testing are excluded from the first slice.

**Considered Options**

- Use a broader machine learning curriculum or multiple textbooks from the start.
- Use ISL Python as the primary source for the first graph.

**Consequences**

V1 graph authoring has a clear source basis and source locator strategy, but the first benchmark will reflect ISL's framing of classical supervised statistical learning.
